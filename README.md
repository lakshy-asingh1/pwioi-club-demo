# PW Institute of Innovation - Web Project

## Project Overview
This project is a comprehensive website for **PW Institute of Innovation**. It showcases various aspects of the institute, including events, clubs, registration forms, gallery, and contact information. The goal is to provide a modern, responsive, and interactive interface for students and visitors.

## Project Description
The PW Institute of Innovation website includes the following features:

1. **Landing Page (`index.html`)**
   - Brief introduction about the institute
   - Navigation to other pages
   - Highlights of upcoming events and featured clubs

2. **Events Page (`events.html`)**
   - Display upcoming events
   - Event registration form

3. **Clubs Page (`clubs.html`)**
   - Showcase all clubs at the institute
   - Links to join clubs

4. **Registration Page (`registration.html`)**
   - Sign-up form for events and memberships
   - Basic client-side validations

5. **Sign-in / Sign-up Pages**
   - Secure login and registration forms
   - Navigation to different parts of the website

6. **Gallery Page (`gallery.html`)**
   - Showcase images from past events

7. **Contact Page (`contact.html`)**
   - Contact form and institute information
